# 🔐 Password Manager (Python)

A simple and secure Password Manager built using Python that helps users generate strong passwords and store login credentials efficiently.

## 🚀 Features
- Generate strong and random passwords
- Save website, email, and password details
- Data stored locally using file handling
- Simple and user-friendly interface

## 🛠️ Tech Stack
- Python
- Tkinter (for UI)
- Random module
- File handling (JSON / Text)

## ▶️ How to Run
1. Install Python
2. Run the main file:
   python main.py
